How It Works
Add Role

Adds the selected role to users who don't already have it. Existing roles are preserved.

Remove Role

Removes the selected role from users who have it. Other roles are preserved.

Set as Only Role

Caution: Removes all existing roles and sets only the selected role.

Quick Tips
  • Use Add to give Observer access to multiple reviewers at once
  • Use Remove to revoke Panel membership after a review cycle
  • Use Set as Only to reset users to just Colleague role
