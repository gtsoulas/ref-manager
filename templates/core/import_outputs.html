Expected CSV Format

Your CSV file should have the following columns:

Column Name Required Description
Title Required Output title
Person Required Author(s), separated by //
Current publication status Optional Published, Accepted/In press, etc.
Type Optional Article, Book, Chapter, etc.
Journal title Optional Name of journal
Publisher Optional Publisher name
DOIs (Digital Object Identifiers) Optional DOI identifier
Full date Optional Publication date
Earliest published date Optional Alternative publication date
Host publication volume Optional Volume number
Issue number Optional Issue number
Pages (from-to) Optional Page range
ISBN (print) Optional ISBN for books
REF Open Access compliance status Optional OA compliance status
Pure ID Optional Stored in notes field
Organisational unit Optional Stored in notes field
Note: The import function will:
  • Automatically parse multiple authors from the Person field (separated by //)
  • Use the first author as the primary staff member for the output
  • Create new staff records if they don't exist (if enabled)
  • Skip duplicate outputs based on title (if enabled)
  • Store additional information (Pure ID, organisational units, all authors) in the notes field
